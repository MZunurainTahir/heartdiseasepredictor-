

# Heart Disease Predictor 🫀

An end-to-end machine learning-based web application designed to predict the likelihood of heart disease using clinical patient data. This tool analyzes various cardiovascular health metrics to assist in early risk assessment and data-driven health insights.

## 📊 Dataset Metrics Included

The model evaluates key physiological parameters, including:
*   Age & Gender
*   Chest Pain Type (CP)
*   Resting Blood Pressure (trestbps)
*   Serum Cholesterol (chol)
*   Fasting Blood Sugar (fbs)
*   Resting Electrocardiographic Results (restecg)
*   Maximum Heart Rate Achieved (thalach)
